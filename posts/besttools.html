Melhores Tecnologias para DEVs

Atualmente, pessoas estão opitando por aprenderem novas tecnologias e linguagens de programação, dito isso, algumas delas não sabem por onde começar e ficam perdidas para iniciar em desenvolvimento WEB. Então vim hoje com essa postagem para deixar mais claro quais são as tecnologias básicas para desenvolvimento web.

HTML, CSS, JAVASCRIPT

De acordo com oMDNessas três são as tecnologias principais básica que todo desenvolvedor WEB precisa saber.

HTML

O HTML é uma linguagem de marcação de hipertexto, que consiste em por diferentes elementos que no qual você pode agrupa-los/marca-los e dar um significado (Semântica). Se parassemos para comparar com uma construção de um edifício, o HTML seria como asfundações e as paredes da casa, que lhe dão estrutura e mantém unida.

CSS

O CSS (Cascading Style Sheets) ela é uma linguagem baseada em regras, também é voltada para aplicar estilos ao HTML e deixa-lo bonito, por exemplo, definir cores de fundo, incrementar bordas, animar as coisas ou até mesmo fazer um layout de uma página de uma determinada maneira. Novamente, comparando com uma casa, o CSS é como a pintura, o papel de parede, os tapetes e as pinturas que você usaria pra deixar essa casa bonita.

JavaScript

O JavaScript ele é uma linguagem de programação que usamos para colocar interações do usuário com o site, desde uma troca dinâmica de estilos até a obtenção de atualizações de servidor ou até mesmo o uso de gráficos 3D bem complexos. Ultima comparação com um edifício, o JavaScript é como o fogão, uma TV, geladeira, microondas ou até mesmo secador de cabelo. E isso são coisas que fornecem uma funcionalidade útil à sua casa.
